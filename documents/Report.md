# Отчет по итогам тестирования
### Краткое описание
Была проведена работа по автоматизации тестирования приложения для оплаты тура.  

Были написаны:
* Позитивные и негативные UI тесты
* Тесты для проверки записей в БД

### Количество тест-кейсов
* Всего: 30 тестов
* Успешных: 24 (80%)
* Неуспешных: 6 (20%)
